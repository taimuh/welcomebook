# テキスト装飾・入力記法ガイド

WelcomeBook のマニュアルやウェルカムメッセージで使える「装飾・レイアウトの記法」をまとめました。
難しい知識は不要です。記号を組み合わせるだけで、見やすく整ったページが作れます。

## 1. 文字の装飾

### 太字

`**テキスト**` のように、テキストを `**` で囲むと太字になります。

```
**重要なポイント**はここです。
```

> **重要なポイント**はここです。

---

### ハイライト（蛍光ペン）

`==テキスト==` のように、テキストを `==` で囲むと黄色でハイライトされます。

```
チェックアウトは ==午前11時まで== です。
```

> チェックアウトは <mark>午前11時まで</mark> です。

---

### 文字の色

`{色:テキスト}` の形で書くと、文字に色がつきます。

| 書き方 | 色 |
|---|---|
| `{red:テキスト}` | 赤 |
| `{blue:テキスト}` | 青 |
| `{green:テキスト}` | 緑 |
| `{orange:テキスト}` | オレンジ |

```
{red:ゴミ出しは必ずルールを守ってください。}
水道の元栓は{blue:青いバルブ}です。
```

> <span style="color:#dc2626">ゴミ出しは必ずルールを守ってください。</span>
> 水道の元栓は<span style="color:#2563eb">青いバルブ</span>です。

## 2. 見出し（タイトル）

`#` の数で見出しの大きさが変わります。`#` のあとに半角スペースを入れてください。

```
# 大見出し（ページタイトル）

## 中見出し（セクション）

### 小見出し（項目）
```

> # 大見出し（ページタイトル）
> ## 中見出し（セクション）
> ### 小見出し（項目）

## 3. 箇条書き・番号リスト

### 箇条書き（順不同）

行の先頭に `- ` （ハイフン＋半角スペース）を付けます。

```
持ち物チェックリスト：

- パスポート
- 現金
- 充電器
```

> 持ち物チェックリスト：
> - パスポート
> - 現金
> - 充電器

---

### 番号リスト（手順の説明に便利）

行の先頭に `1. ` のように番号を付けます。番号は自動で振られるので、全部 `1.` でも大丈夫です。

```
チェックインの手順：

1. 玄関のキーボックスに暗証番号を入力する
1. 鍵を取り出す
1. 部屋のドアを開ける
```

> チェックインの手順：
> 1. 玄関のキーボックスに暗証番号を入力する
> 1. 鍵を取り出す
> 1. 部屋のドアを開ける

## 4. リンク

`[表示するテキスト](URL)` の形で書きます。リンクは自動的に新しいタブで開きます。

```
詳しくは[アクセスマップ](https://maps.google.com/)をご確認ください。
```

> 詳しくは[アクセスマップ](https://maps.google.com/)をご確認ください。

## 5. 画像の挿入

### 基本の書き方

`![説明文](画像のURL)` の形で書きます。
「説明文」は画像が表示されなかったときに代わりに表示されるテキストです。

```
![エントランスの写真](https://example.com/entrance.jpg)
```

---

### サイズを指定する

URL の後ろに `"横px x 縦px"` を付けると、表示サイズを指定できます。

| 書き方 | 意味 |
|---|---|
| `"600x400"` | 横600px・縦400px |
| `"600x"` | 横600px（縦は自動） |
| `"x400"` | 縦400px（横は自動） |

```
![エアコンリモコン](https://example.com/remote.jpg "400x")
```

> ※ 画面の幅より大きなサイズを指定しても、はみ出しません。

---

## 6. 画像を横に並べる

複数の画像を横並びで表示したいときは、`::: image-row` と `:::` で画像を囲みます。

```
::: image-row
![手順1：ボタンを押す](https://example.com/step1.jpg "300x200")
![手順2：ランプを確認する](https://example.com/step2.jpg "300x200")
:::
```

- 画像の数に制限はありません
- スマートフォンでは画面幅に合わせて自動で折り返されます
- サイズ指定と組み合わせると、均等に並べやすくなります

## 7. 表（テーブル）

`|` （縦棒）と `-` （ハイフン）を使って表を作れます。

```
| 項目 | 内容 |
|---|---|
| Wi-Fiのパスワード | welcome2024 |
| チェックイン | 15:00〜 |
| チェックアウト | 〜11:00 |
```

> | 項目 | 内容 |
> |---|---|
> | Wi-Fiのパスワード | welcome2024 |
> | チェックイン | 15:00〜 |
> | チェックアウト | 〜11:00 |

ヘッダー行（1行目）が不要な場合は、最初の `|---|` の行から始めることもできます。

```
|---|---|
| Wi-Fiのパスワード | welcome2024 |
| チェックアウト | 〜11:00 |
```

## 8. よくある質問

**Q. 記号を文章中に普通に使いたい場合はどうすればいいですか？**
A. `\` （バックスラッシュ）を記号の前に付けると、そのまま表示されます。
例：`\*\*太字にならない\*\*` → \*\*太字にならない\*\*

---

**Q. 改行したのに1行にまとまってしまいます。**
A. 段落を分けたいときは、間に **空白行を1行** 入れてください。

```
1段落目のテキストです。

2段落目のテキストです。（空白行で区切る）
```

---

**Q. 半角スペースが必要な場面はどこですか？**
A. 以下の場所では **`#` や `-` や `1.` のすぐ後ろに半角スペース** が必要です。

```
# 見出し    ← # の後ろにスペース
- 箇条書き  ← - の後ろにスペース
1. 番号     ← . の後ろにスペース
```
